Bridging the gap between humans and computers

We want to change how people interact with their computers. Natural language interfaces will be the future of interacting with applications and devices. Often writing the code that implements the logic of an application is the easy bit. Finding a way for people to efficiently interact with it is the hard part. Inimda allows requests in natural language to be translated accurately into code, using the existing apis of an application, that can then be immediately run.

We've all found ourselves asking Google "how do I do X in Y?", sometimes we get lucky, but maybe we get results full of blog-posts full of spam, or just a Stack-overflow question with an out-dated answer. In any case, using an application in a way we haven't before is often painful and slow. It's annoying, but also destructive, people end up not being able to do what they want.

We want to change this. Advances in AI models, computation speed, and massive text corpora are making some amazing things possible. Inimida reads the documentation for you, knows all the APIs, has gone through all the examples. It listens to what you want to do, writes code to that, and runs it.

People talk, it's how we interact with other people, cats, innanimate objects, to varying degrees of success. Even if it doesn't work, it is how we want to interact. Let's make it work like that for computers.
